abbr 태그: 줄임말을 표시하고 title속성을 같이 사용할 수 있음
예시: abbr 태그 확인용

cite 태그: 책이나 음악, 영화, 예술 작품 등과 같은 창작물의 제목이나 참고 내용 표시
예시: cite 태그 설명은 TCP SCHOOL에서 참고

code 태그: 컴퓨터 인식을 위한 소스 코드
예시: System.out.println("code 사용 방법_자바코드 일부");

small 태그: 텍스트 작게 표시
예시: 원래_Small 태그사용_Small

sub 태그: 아래 첨자 표시
예시: 아래 첨자아래 첨자

sup 태그: 위 첨자 표시
예시: 위 첨자위 첨자

s 태그: 취소선 표시
예시: 취소선 표시

u 태그: 밑줄 표시
예시: 밑줄 표시

ins 태그: 공동 작업 문서에서 새로운 내용 삽입
예시: + 다음 새로운 내용 삽입 + 새로운 내용

del 태그: 공동 작업 문서에서 기존 내용 삭제
예시: + 다음 새로운 내용 삭제 + 새로운 내용
